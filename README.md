# Argon V3 Configuration Script

## Description

This script automates the process of configuring your Raspberry Pi for use with the Argon V3 case, supporting both the normal version and the version with NVMe. It applies the necessary `config.txt` settings for optimal operation, including overclocking, GPIO, UART, and other required configurations. Additionally, it supports the Argon V3 DAC for audio output.

## Features

- Configures the Raspberry Pi for use with Argon V3 normal or Argon V3 with NVMe.
- Updates `config.txt` with essential parameters such as GPIO, I2C, UART, and USB current limits.
- Configures the Raspberry Pi to support the Argon V3 DAC.
- Provides an easy-to-use menu for selecting configurations based on the Argon V3 version.

## Requirements

- A Raspberry Pi with Raspberry Pi OS or compatible system.
- Argon V3 case (normal or with NVMe).
- No special permissions are required to run this script.
